# Pattern: D8 Prompt Markers

## 문제

프롬프트 템플릿 내에서 서브에이전트의 동작을 제어하거나, 특정 상황에서만 지식을 주입해야 할 때 발생하는 제어의 어려움.

## 해법

D8 마커(예: `/* primary-only */`)를 사용하여 프롬프트 템플릿의 특정 섹션을 조건부로 활성화하거나 비활성화합니다.

```typescript
// 예시: 프롬프트 템플릿 내 마커 사용
/* primary-only */
// 이 섹션은 프라이머리 에이전트에게만 주입됩니다.
```

## 사용 시점

- 프롬프트 주입 시 서브에이전트와 프라이머리 에이전트의 컨텍스트를 분리해야 할 때
- 특정 지식 노드가 특정 에이전트에게만 필요할 때

## 사용하지 말 것

- 지원되지 않는 마커를 사용하거나, 마커의 의미를 오해하여 잘못된 컨텍스트를 주입하는 경우.
